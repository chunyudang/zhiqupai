# 识趣派 — 产品整体功能规划图 v2.0

> "轻学习+强激励"的答题闯关娱乐APP

---

## 一、产品定位

**一句话**：把刷手机的碎片时间，变成涨知识的游戏化体验。

**目标用户**：大众泛用户

**核心体验**：用户在 **5分钟内** 完成一轮答题闯关，通过通识知识题目提升理性思维与综合认知能力，获得积分奖励和成就感。

**MVP端限制**：MVP 阶段仅支持 **微信小程序端**（基础库 3.0.0+，微信 iOS 8.0+ / Android 8.0+）

**客户端框架**：**uni-app**（一套代码，后续可扩展 iOS / Android / 鸿蒙）

## 三、用户激励体系

```
答题闯关 → 获得积分 → 积分用于：
                        ├── 积分商城兑换（消耗积分→获得商品）
                        ├── 有奖竞猜下注（消耗积分→赢更多积分）
                        └── 签到补签（消耗50积分/次）
```

### 答题计分规则

| 规则项 | 值 |
|--------|-----|
| 每题答对 | +15分 |
| 通关条件 | 答对 ≥ 4/6（正确率 ≥ 67%） |
| 通关奖励 | 额外 +50分 |
| 满分一关 | 6×15 + 50 = **140分** |
| 失败情况 | 仅获得已答对题目的分数 |
| 重复答题 | **0分**（已通关关卡再次答题不得分） |

### 签到规则

| 规则项 | 值 |
|--------|-----|
| 每日积分 | 第1天+1、第2天+2…第7天+7，第8天起固定+7 |
| 周期 | 自然月重置 + 断签重置 |
| 补签 | 消耗50积分/次，月限5次，后台可配置 |

## 四、核心用户路径

```
访问APP → 底部Tab导航

MVP期间Tab（5个）：
  ├── 🏠 首页 → 选择学科 → 选择关卡 → 答题闯关 → 结果/积分
  ├── 🎯 竞猜 → 提示「功能完善中……」 
  ├── 🛒 商城 → 积分商城首页 → 商品详情 → 兑换
  ├── 💬 消息 → 各类型消息查看
  └── 👤 我的 → 个人中心（签到/数据/记录/明细/设置）
```

## 五、北极星指标体系

| 指标 | 目标 |
|------|------|
| 次日留存 | ≥ 40% |
| 答题完成率 | ≥ 70% |
| 关卡通关率 | ≥ 50% |
| 用户日均答题 | ≥ 2轮 |
| 积分消耗率 | ≥ 30% |（含补签+商城兑换） |
| 商城兑换率 | ≥ 10% | 拥有积分的用户中完成过至少一次兑换的比例 |
| 竞猜参与率（P1后） | ≥ 20% |
